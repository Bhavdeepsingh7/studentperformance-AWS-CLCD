# 🎓 Student Performance Predictor

A data-driven web application that predicts student academic performance and provides actionable insights using analytics and machine learning.

---

## 🚀 Overview

The **Student Performance Predictor** helps analyze how various factors (study time, attendance, socio-economic background, etc.) influence student performance.

Instead of just predicting scores, this project focuses on:

* 📊 **Data Analytics**
* 🤖 **Machine Learning Predictions**
* 🧠 **Insight Generation**

---

## 🎯 Problem Statement

Educational institutions often struggle to:

* Identify at-risk students early
* Understand key factors affecting performance
* Take data-backed decisions

This project solves that by combining **predictive modeling + analytics dashboards**.

---

## ✨ Features

### 🔮 1. Performance Prediction

* Predict student scores based on input features
* Uses trained ML models (e.g., Linear Regression / Random Forest)

---

### 📊 2. Analytics Dashboard

* Visualize:

  * Score distribution
  * Study time vs performance
  * Attendance impact
* Helps uncover trends and patterns

---

### 🧠 3. Insight Generation

* Automatically generates insights like:

  * “Students studying >3 hours/day score 20% higher”
  * “Attendance below 75% strongly impacts performance”

---

### ⚠️ 4. Risk Detection

* Identifies students likely to underperform
* Enables early intervention

---

### 📈 5. Feature Importance Analysis

* Shows which factors most influence performance
* Helps educators focus on key areas

## ⚙️ How It Works

1. User inputs student data
2. Backend processes input
3. ML model predicts performance
4. Analytics module:

   * Aggregates data
   * Generates visualizations
   * Produces insights

---

## 📊 Analytics Focus (Key Highlight)

This project goes beyond prediction:

* **Exploratory Data Analysis (EDA)**

  * Correlation heatmaps
  * Distribution analysis

* **Trend Analysis**

  * Study habits vs scores
  * Attendance vs performance

* **Comparative Insights**

  * High vs low performers

* **Data-Driven Recommendations**

  * Personalized suggestions for improvement
